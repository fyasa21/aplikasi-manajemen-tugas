# Actask - Task Management Web App

Actask adalah aplikasi manajemen tugas berbasis web yang dirancang untuk membantu pengguna mengelola aktivitas harian secara sederhana dan efisien.

Aplikasi ini menggunakan konsep Kanban Board untuk mengelompokkan tugas ke dalam beberapa status seperti:
- Todo
- In Progress
- Done

Project ini dibuat sebagai bagian dari praktikum pengembangan aplikasi berbasis web dan kolaborasi menggunakan Git.

## Teknologi yang digunakan

- HTML
- CSS
- JavaScript (Vanilla JS)
- LocalStorage
- Progressive Web App (PWA)
- Service Worker
- Git & GitHub

## Tujuan Sistem

1. Membantu pengguna mencatat dan mengelola tugas.
2. Menyediakan tampilan visual menggunakan Kanban board.
3. Memungkinkan penggunaan aplikasi secara offline melalui PWA.
4. Menerapkan workflow kolaborasi menggunakan Git.
