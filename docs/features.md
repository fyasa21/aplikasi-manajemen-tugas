# Fitur Aplikasi

## 1. Manajemen Task
Pengguna dapat:

- Menambahkan task
- Mengedit task
- Menghapus task
- Mengubah status task

## 2. Kanban Board
Task ditampilkan dalam tiga kolom:

- Todo
- In Progress
- Done

Task dapat dipindahkan menggunakan fitur **drag and drop**.

## 3. Authentication
Sistem login sederhana menggunakan LocalStorage.

Fitur:
- Register
- Login
- Logout
- Session checking

## 4. Dashboard
Dashboard menampilkan ringkasan:

- Total task
- Task selesai
- Task pending

## 5. Progressive Web App (PWA)
Aplikasi dapat berjalan secara offline menggunakan:

- manifest.json
- service worker

## 6. Offline Support
Jika koneksi internet terputus, aplikasi tetap dapat diakses karena asset telah disimpan di cache browser.
